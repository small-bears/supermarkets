#### 三、 项目介绍

基于SSM的超市管理系统

角色：管理员、会员、员工、供应商

管理员：管理员点击登录摁钮页面跳转到金福超市管理系统对系统首页、会员管理、员工管理、供应商管理、商品信息管理、进货记录管理、配送记录管理、损坏替换管理、损坏退货管理、商品库存管理、销货记录管理、补货记录管理、商品类型管理、留言板管理、系统管理、我的信息等功能，进行相应操作，

会员：当用户打开系统的网址后，首先看到的就是首页界面。在这里，用户能够看到金福超市管理系统的导航条显示系统首页、商品信息、促销公告、留言板、个人中心、购物车等。用户点击个人中心，在个人中心页面可以更新个人详细信息，还可以对修改密码、我的订单、我的地址、我的收藏进行详细操作；

员工：员工进行系统后，主要包括对系统首页、会员管理、供应商管理、商品信息管理、进货记录管理、配送记录管理、损坏替换管理、损坏退货管理、商品库存管理、销货记录管理、补货记录管理、订单管理、我的信息等功能进行操作。

供应商：供应商进行系统后，主要包括对系统首页、进货记录管理、配送记录管理、损坏替换管理、损坏退货管理、我的信息等功能进行操作。

#### 四、项目技术

后端：SSM(Spring+SpringMVC+Mybatis)

前端：Vue +ElementUI

#### 五、 运行环境

开发语言：Java

开发工具：IDEA，Eclipse,Myeclipse都可以。推荐IDEA

JDK版本：1.8

数据库: MySQL5.7

Tomcat环境：Tomcat 7.x,8.x,9.x版本均可

项目管理：Maven



#### 六、安装教程

使用Navicat或者其它工具，在mysql中创建对应名称的数据库，并导入项目的sql文件；

使用IDEA/Eclipse导入项目，若为maven项目请选择maven;导入成功后请执行maven clean;maven install命令，然后运行；

进入src/main/resources修改application.yml里面的数据库配置

启动项目
